# PhonePe Analytics Dashboard 📊

## 📌 Overview
This project presents a **Power BI dashboard** analyzing PhonePe digital payment data.  
It provides insights into transaction volumes, values, user demographics, and service usage patterns, helping visualize the growth of digital payments in India.

## 🎯 Key Features
- **KPIs & Metrics**: Total transactions, transaction value, unique users, and success rate.  
- **User Demographics**: Age segment contribution (Gen X, Millennials, Gen Z, Boomers) and weekday vs. weekend usage.  
- **Service Analysis**: Transaction distribution across UPI, QR, mobile transfers, loans, and other services.  
- **Top Users**: Identification of high‑value users and their transaction activity.  
- **Performance Trends**: Month‑over‑month growth, success vs. failed transactions, and comparative service values.  

## 🛠️ Tools & Technologies
- Power BI  
- DAX (Data Analysis Expressions)  
- Data Modeling  
- Visualization Design
